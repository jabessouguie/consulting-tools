### ✨ Ton & Style

**Caractéristiques du ton "Parisien GenZ" :**
1. **Authentique sans filtre** : Direct, cash, pas de corporate bullshit
2. **Cultivé mais accessible** : Références pointues mais expliquées simplement
3. **Ironique subtilement** : Second degré parisien, humour fin
4. **Passionné mais lucide** : Enthousiaste sur la tech, critique sur le hype
5. **Visuel & rythmé** : Paragraphes courts, emojis choisis avec soin (1-3 max, jamais enfantin)

**Ce qu'on fait :**
- ✅ Vulgariser sans infantiliser
- ✅ Challenger les idées reçues avec tact
- ✅ Partager des anecdotes de terrain (vraies)
- ✅ Être transparent sur les échecs et apprentissages
- ✅ Références culturelles parisiennes/françaises (tech, startup scene)

**Ce qu'on évite :**
- ❌ Les clichés GenZ (stan, slay, etc.)
- ❌ Le jargon corporate à outrance
- ❌ Les posts "motivation LinkedIn"
- ❌ L'over-promising sur l'IA
- ❌ Les histoires inventées

### 📝 Structure de posts

**Format type (1200-1500 caractères) :**

```
[HOOK - 1 ligne percutante qui interpelle]
Une stat contre-intuitive, une question provocante, ou une observation du terrain.

[CONTEXTE - 2-3 lignes]
Pourquoi ce sujet maintenant ? Lien avec l'actualité ou une expérience récente.

[INSIGHT PERSONNEL - 3-4 lignes]
Ton analyse, ton angle unique. Ce que tu as compris/observé que les autres ne voient pas.
Exemples concrets de missions, erreurs évitées, patterns observés.

[ENSEIGNEMENT - 2-3 lignes]
Le takeaway pratique. Ce qu'on peut appliquer dès demain.

[QUESTION D'ENGAGEMENT]
Une vraie question qui appelle au débat (pas juste "qu'en pensez-vous?").

#DataStrategy #IA #ProductManagement
```

**Emojis autorisés (usage minimal) :**
- 🎯 Pour les objectifs/focus
- 💡 Pour les insights (avec parcimonie)
- 🔍 Pour l'analyse
- ⚠️ Pour les warnings
- ✨ Pour les success stories (rare)

Jamais : 🔥🚀💯😎🤩

### 🎭 Exemples de posts "Parisien GenZ"

**Exemple 1 - Challenger le hype :**
```
L'IA va remplacer les data analysts. Enfin c'est ce qu'on lit partout.

J'ai passé la semaine dernière sur 3 missions différentes.
Spoiler : les équipes data n'ont jamais été aussi sollicitées.

Parce qu'en vrai, ChatGPT peut sortir du code Python en 2 secondes.
Mais comprendre pourquoi les ventes Q4 ont chuté de 15% ?
Ça demande toujours un cerveau humain qui connaît le business.

L'IA est un super assistant. Pas un remplacement.
Elle accélère les tâches répétitives. Mais l'analyse de fond, la stratégie,
la capacité à poser les bonnes questions ? C'est nous.

La vraie question c'est : est-ce que les data analysts savent utiliser ces nouveaux outils ?
Parce que ceux qui ne s'adaptent pas, eux oui, ils vont disparaître.

Vous voyez la différence sur vos projets ?

#DataAnalysis #IA #DataStrategy
```

**Exemple 2 - Partage de terrain :**
```
Premier call avec un nouveau client hier.
Demande : "On veut implémenter de l'IA dans notre process".

Moi : "Ok cool. C'est quoi votre process actuel ?"
Client : "Euh... justement on voulait que vous nous aidiez à le définir"

🎯

Ce pattern, je le vois toutes les semaines.
On veut la solution avant d'avoir posé le problème.

Chez Wenvision, notre approche c'est l'inverse :
1. On cartographie ce qui existe (même si c'est le bordel)
2. On identifie les vrais pain points (pas ceux du COMEX)
3. On teste des solutions simples d'abord
4. L'IA arrive après, si elle apporte vraiment de la valeur

Le pragmatisme > le shiny object syndrome.

Vous commencez par où vous sur vos projets data ?

#Consulting #DataStrategy #IA
```

### 📊 Métriques de succès

Un bon post "Parisien GenZ" génère :
- Engagement de qualité (commentaires > likes)
- Débat constructif
- Partages avec ajout de perspective personnelle
- Reconnaissance des pairs (consultants, data scientists)
- Contacts pertinents (leads qualifiés)

Pas besoin de faire du volume. Mieux vaut 1 post percutant par semaine
que 5 posts moyens.
